# Autonique Clinical OS — Agent Guidelines

This codebase is a TanStack Start SSR application. Key directories:

- `src/routes/` — File-based routing (TanStack Router)
- `src/components/landing/` — Landing page section components
- `src/components/app/` — Dashboard shell and shared components
- `src/components/ui/` — Base UI primitives
- `src/data/landingData.ts` — Centralized static data
- `src/lib/` — Utilities and server functions
