# Autonique Clinical OS

A premium SaaS platform for modern clinic operations — bookings, patients, billing, and team management in one calm workspace.

## Stack

- **Framework**: TanStack Start (SSR)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript + React 19
- **Charts**: Recharts
- **Payments**: Stripe Sandbox
- **Deploy**: Vercel (Nitro Vercel preset)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Build

```bash
npm run build
```
